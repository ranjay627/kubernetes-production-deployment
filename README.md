# Kubernetes Production Deployment

This project demonstrates a production-grade Kubernetes deployment of a Spring Boot application.

## 🚀 Features

- Dockerized Spring Boot application
- Kubernetes Deployment & Service
- Ingress for external access
- ConfigMap & Secret management
- Horizontal Pod Autoscaling (HPA)

## 🧱 Architecture

User → Ingress → Service → Pods

## 📦 Tech Stack

- Spring Boot
- Docker
- Kubernetes
- GitHub

## 📌 Goal

To demonstrate real-world Kubernetes deployment skills with production-level practices.
